STAT2 Combinatorial Library Analysis

Analysis of a combinatorial library of STAT2 mutants. All possible combinations of human and mouse STAT2 amino acids 198-209 range were generated in a yeast-2 hybrid system and assessed for binding by flaviviral NS5 protein. Experiments by lab of Matt Evans, analysis by lab of Jesse Bloom.

  • Scatterplot of all sequences - highlight mouse mutations, filter based on standard dev
